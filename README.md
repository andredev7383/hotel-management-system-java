# Hotel Management System

## Sobre
Projeto simples em Java para gerenciamento de quartos de hotel.

- Cadastro de hóspedes
- Check-in
- Check-out
- Listagem de quartos

## Tecnologias

- Java
- Programação Orientada a Objetos

## Como executar

1. Clone o repositório
2. Compile os arquivos
3. Execute Main.java
